# Игровые механики

## Концепция
Фентезийная карточная игра с механикой путешествий между мирами через порталы. Игрок собирает колоды, побеждает врагов, получает награды и улучшает свою крепость.

## Прогресия
- [ ] Валюта (души/золото/мана) за победы
- [ ] Сундуки с картами, рубашками, улучшениями
- [ ] Коллекция рубашек карт
- [ ] Система достижений

## Рогалик элементы
- [ ] Случайные порталы в разные миры
- [ ] Каждый мир = уникальные враги, правила, награды
- [ ] Редкость миров (обычный → редкий → легендарный)

## Несколько колод
- [ ] Открытие новых колод за достижения
- [ ] Каждая колода — свой архетип (агро, контроль, мидрейндж)
- [ ] Возможно комбинирование колод

## База/строительство
- [ ] Своя крепость/замок стоительство
- [ ] В зависимости от уровня постройки - увеличивается: количество карт, колод, рубашек, ресурсов
- [ ] НПС торговцы, кузнецы - только в мирах в порталах

## Вызовы
- [ ] Энкаунтеры с возрастающей сложностью
- [ ] Бесконечные волны
- [ ] Ежедневные/недельные испытания с ротацией
- [ ] Турниры

## Коллекционирование
- [ ] Полная коллекция всех карт
- [ ] Сезонные события
- [ ] Асинхронный мультиплеер (дуэли с призраками других игроков)

## Классификация/Свойства карт
- [] Тип карты: Уникальная, Базовая
- [] Свойство: Фракция (люди, гномы, эльфу, животные, магические существа и т.п.)
- [] Свойство: Количество - Разрешенное количество в колоде
- [] Свойство: Скилл - Бафф, Прокрутка и т.п - важно. Скилл работает только на свою фракцию или на все карты

## Классификация/Свойства врагов
- [] Продумать защиту от карт фракции
- [] Уникальная способность - ослабляет карты, уменьшает урон, уменьшает сбросы, уменьшает ходы, отключает способности, деактивирует тип карт

## Разные рубашки карт
- [] Дают: рубашку, количество ходов, количество сбросов, количество карт в руке, могут баффать карты (либо все, либо по фракции или типу карты)
