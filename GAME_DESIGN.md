# Игровые механики

## Концепция
Фентезийная карточная игра с механикой путешествий между мирами через порталы. Игрок собирает колоды, побеждает врагов, получает награды и улучшает свою крепость.

---

## Сюжетная система

### Режимы прохождения

#### 1. Обучение (Tutorial)
- Обязательная первая миссия — 1 портал для прохождения
- После прохождения: стартовая дека людей + полностью открытый сюжет людей
- Цель: познакомить игрока с механикой боя

#### 2. Рогалик (Roguelike)
- 6 слотов под порталы
- Порталы "растут" по таймеру (1 портал за 2 часа реального времени)
- За кристаллы можно создать уникальный портал мгновенно
- Босс: появляется после заполнения всех 6 слотов
- Награда за босса: случайная карта (редко — unique)

#### 3. Сюжетные порталы (Story Portals)
- Открываются при получении карты новой расы
- Дополнительный портал с историей этой расы
- Прохождение сюжета расы → уникальные карты этой расы

### Прогрессия

```
База
 │
 ├── Обучающая миссия (1 портал)
 │   → Стартовая дека людей
 │   → Сюжет людей открыт
 │
 └── Рогалик (6 слотов)
      │
      ├── Порталы растут по таймеру (1/2ч)
      │   ├── 80% — случайный враг/карта
      │   └── 20% — карта новой расы → сюжетный портал
      │
      ├── Босс (когда слоты заполнены)
      │   └── Награда: случайная карта (редко unique)
      │
      └── Кристаллы → создать портал сразу
```

## Открытые вопросы ( требуют проработки )

### 1. Открытие новой расы ✓
**Решение:** 1 карта = новый сюжетный портал

- Если игрок получает (покупает, выигрывает, получает в подарок) карту расы, которой у него ещё нет — открывается сюжетная линия этой расы
- Сюжетный портал появляется на карте как дополнительный (поверх 6 слотов рогалика)
- Прохождение сюжета расы даёт:
  - Уникальные карты этой расы
  - Глубокое погружение в лор
  - Доступ к колоде этой расы

### 2. Структура сюжета расы ✓
**Решение:** Последовательное прохождение карты с предустановленными врагами

- Сюжет расы = карта с несколькими узлами (эпизодами)
- Каждый узел = враг + диалог перед боем
- Последовательное прохождение (1 → 2 → 3 → босс)
- Босс в конце карты выдаёт 1 случайную карту этой расы
- Диалоги — повествовательные, без выбора

### 3. Босс и награды ✓
**Решение:** Босс — последний enemy на карте сюжета

- Босс всегда последний enemy на карте (финальный узел)
- Награда босса: случайная карта РАСЫ сюжета, которой ещё нет у игрока
- Пример:
  - Нет эльф/Лучницы → выдаст её
  - Потом нет эльф/Паладин → выдаст его
  - Потом любую эльфийскую карту, которой нет
- Если все карты расы уже есть — сюжет пройден полностью, босс не появляется

### 4. Кристалы — источники ✓
**Решение:** Награды за победы + экономика

- Текущий баланс наград: 75% золото / 25% кристаллы
- Планируется: 90/10% или 99/1% (точная экономика будет прописана позже)
- Кристаллы используются для:
  - Создание unique портала (мгновенно, без ожидания таймера)
  - Unique портал → босс гарантированно выдаёт unique карту

### 5. Уникальные (Unique) карты ✓
**Решение:** Особые карты с уникальными способностями

- Обладают уникальными способностями (не доступны обычным картам)
- Максимальное количество в колоде: 1 штука
- Не открывают сюжетные линии
- Получение:
  - Unique портал за кристаллы (гарантировано)
  - Редкий шанс с обычного боса

### 6. Торговцы ✓
**Решение:** Рандомные торговцы на рогалик-картах

- Появляются случайным образом на рогалик-картах
- Шанс появления: ~25-15% (1/4 — 1/6)
- Расположение: специальная ячейка на карте в портале
- При клике: открывается модальное окно
- Продают:
  - Карты (только те, которые игрок уже открыл/имеет)
  - Рубашки карт
  - Артефакты (в будущих релизах)
- Валюта: золото
